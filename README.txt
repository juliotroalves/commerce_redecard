# Drupal Commerce RedeCard module Integration

This module provides a way of integration to Drupal Commerce payments with
RedeCard WebService of credit cards.

Installation
------------

To install Redecard module payments enable redecard module and go to 
admin/commerce/payment-methods. Enable rule that active this module and 
configure with your redecard account data, to use on checkout page.
